# CLAUDE.md
## プロジェクト概要
キャンプ記録サービスを作成
みんなのキャンプ記録が見たいのと、おすすめのキャンプ情報が分散しているのでこのサービスで集約したい
Instagramにみんなキャンプログを投稿するので、Instagramにシェアしやすい形を取りたい

## 技術スタック
- Next.js (App Router)
- Tailwind CSS
- Google Maps API（マップピン機能）

## デザイン方針
- Maplyのようなモダンなイメージ
- Poolsuite FMみたいな遊び心あるデザイン
- bumpのような遊び心あるデザインとギミックのイメージ
- wanderoutのような自然系・アウトドア感も取り入れる

## 現在のデザイン実装
- **カラーパレット**: `--forest` (深緑), `--lime` (ライム), `--cream`, `--sand`, `--mist`, `--brown`, `--bark`
- **フォント**: Syne (見出し・ロゴ) + Noto Sans JP (本文)
- **Hero**: Unsplash フルスクリーン背景写真（テント＆森林）+ ダークオーバーレイ + 下部に3層の山シルエットSVG（クリームセクションへのトランジション）
- **フォトストリップ**: ヒーロー直下に横スクロールするキャンプ写真5枚（Unsplash）
- **ヘッダー**: 森緑の固定ナビ + ライムのマーキューバナー（遊び心）
- **CTA**: ライム色の丸ボタン（おもちゃ感を担う）
- 遊び心（マーキュー・ライムアクセント）と自然感（実写キャンプ写真・山シルエット）を両立させる
- 写真素材: `images.unsplash.com`（next.config.ts の remotePatterns に登録済み）
